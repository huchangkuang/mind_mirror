## ADDED Requirements

### Requirement: 系统支持白名单管理员删除攻击性评论
系统 MUST 提供评论删除能力，仅白名单管理员可执行删除操作；初始白名单 MUST 包含账号 `15967537583`。

#### Scenario: 白名单管理员删除评论
- **WHEN** 白名单管理员对目标评论执行删除操作
- **THEN** 系统 SHALL 删除该评论并使其不再出现在普通评论列表中

#### Scenario: 非白名单用户尝试删除评论
- **WHEN** 非白名单用户请求删除任意评论
- **THEN** 系统 SHALL 拒绝请求并返回无权限错误

### Requirement: 删除操作需服务端鉴权与审计记录
系统 MUST 在服务端校验删除操作者身份与白名单权限，并 SHALL 记录删除操作审计信息以支持后续治理追踪。

#### Scenario: 删除请求鉴权通过并记录日志
- **WHEN** 合法白名单管理员成功删除评论
- **THEN** 系统 SHALL 记录操作者标识、评论标识与操作时间

#### Scenario: 未登录请求删除
- **WHEN** 未登录用户发起删除评论请求
- **THEN** 系统 SHALL 拒绝请求并返回登录态相关错误

### Requirement: 反馈评论相关请求 MUST 迁移至外部 API
前端反馈模块的评论列表、发布、点赞与删除请求 MUST 通过外部 Node API 地址调用，并保持原有可见行为与权限语义。

#### Scenario: 获取评论列表
- **WHEN** 前端调用外部 API 获取评论列表（含排序参数）
- **THEN** 系统 SHALL 返回可被现有反馈页直接消费的数据结构
- **THEN** 列表排序与展示行为 SHALL 与迁移前一致

#### Scenario: 发布与点赞评论
- **WHEN** 前端通过外部 API 发布评论或执行点赞
- **THEN** 系统 SHALL 保持原有成功与失败返回语义
- **THEN** 前端交互反馈（数量、状态） SHALL 可正常更新

#### Scenario: 删除评论权限校验
- **WHEN** 前端通过外部 API 请求删除评论
- **THEN** 系统 SHALL 继续按既有权限规则处理
- **THEN** 未授权删除 SHALL 返回明确错误而不是静默失败
